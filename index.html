MediPort:

The Mediport system is an intelligent monitoring solution that helps maintain the effectiveness of vaccines based on their storage conditions during logistics. Certain medicines must be kept in a specific temperature range, which is essential for their effectiveness. While transporting the medicines from the factories to the final destination, the Mediport device will provide us with the temperature reading of the in-built refrigerator, which is then connected to the internet so that the admin can track the container's location. This real-time monitoring system can also be provided to the users as a simple mobile application. In addition to that, The device also functions offline by storing the data in the memory when it gets disconnected, and as soon as the system is online, the stored data is sent to the cloud. The Artificial Intelligence system that runs on the cloud also monitors the shipments' continuous track and detects any supply chain flaw that could lead to a loss in the effectiveness of the medicines. These flaws detected by AI can then be addressed on time, preventing further complications and ensuring the smooth and effective functioning of the medicine procurement process.

This project advanced our team to the National Finals of the Microsoft Imagine Cup-2021.

KLON: Smart Mirror:

KLON was an Internet of Things-based smart home system. It allowed users to access information as well as control various smart home devices. All the relevant information such as the time and date, weather, warning, traffic, music, email, etc was available to the user at a glance. The mirror utilized Amazon Alexa Voice services(AVS) as a medium of interaction between the user and the system. The users could also change the themes of the mirror using the mobile app. The mirror was powered by a Raspberry Pi 3b+ that was linked to various sensors, a speaker, and a microphone. Overall, The mirror was helpful to the users as they could manage their daily activities more easily with the system, while managing their routine house chores.

Path Mapping and Enhanced Line Following on Pololu 3pi+:

The project utilized an event-based way-point creation system to map the task environment while line-following and use this data to improve the line-following performance on subsequent traversals. The systems performance was gauged and analyzed using 2 parameters that are travel time and absolute trajectory error percentage. With the improved system there was an improvement in efficiency of approximately 33.63% across different maps on an average.
